# 6단계 — 기하 작도 확장 ← **완료**

- `[x]` 6-1 점 — **교점** · **대상 위의 점** (라인 트레이서는 stub 유지)
- `[x]` 6-2 선 — **주어진 길이 선분** · **반직선** · **각 이등분선** · **접선** · **벡터**
- `[x]` 6-3 원 — **컴퍼스** · **세 점** · **중심+반지름** · **부채꼴** · **활꼴**
- `[x]` 6-4 다각형 — **정다각형(한 변)** · **정다각형(중심)** · **주어진 크기의 각**

**6-1 조작**
- 교점(I): 선·원 등 도형 2클릭 → 교점(원–원 최대 2개). 부모 이동 시 재계산
- 대상 위의 점(O): 도형 클릭 → 위치 클릭. `POINT_ON` 종속 점
- 라인 트레이서: stub (애니메이션은 이후)

**6-2 조작**
- 반직선: 시작점 → 방향점 (한쪽으로만 연장)
- 벡터: 시작점 → 끝점 (화살표)
- 각 이등분선: A → 꼭짓점 B → C
- 접선: 원 → 점 (원 위 1개 / 바깥 최대 2개)
- 주어진 길이 선분: 시작점 → 길이 입력 → 방향 클릭

**6-3 조작**
- 컴퍼스: 반지름 두 점 → 새 중심
- 세 점 원: 점 3개 (일직선 불가)
- 중심+반지름: 중심 → 숫자/슬라이더 이름 입력
- 부채꼴: 중심 → 끝점1 → 끝점2
- 활꼴: 끝점2 → 호 위 점 (호와 동일)

**6-4 조작**
- 정다각형(한 변): 변의 두 점 → n 입력 → 마우스 방향 클릭
- 정다각형(중심): 중심 → 꼭짓점 → n 입력
- 주어진 크기의 각: 변 위 점 → 꼭짓점 → 각도(도) → 방향 클릭
