## 좌표·드래그와 popscale `factor`

`#wrap`은 `FORTEACHERCD.responsive` / `popscale.js`로 **화면 크기에 맞게 확대·축소**됩니다.  
알지오 UI는 설계 해상도 **1920×1020** 기준이므로, **캔버스 밖 UI 드래그**나 **화면 픽셀(`clientX`/`getBoundingClientRect`)을 레이아웃 좌표로 쓸 때**는 반드시 **factor**를 참고합니다.

```javascript
// 공통 패턴 (common/js/dragContents.js, dragquiz_sci.js 등과 동일)
var factor = FORTEACHERCD.responsive.baseContainerSize.zoom;
var designDx = (e.clientX - startClientX) / factor;
var designLeft = (elementRect.left - parentRect.left) / factor;
```

| 구분 | 좌표계 | factor |
|------|--------|--------|
| 캔버스 `offsetX` / `e.offsetX` | 캔버스 내부 픽셀 (렌더러) | **불필요** |
| 가이드 패널 드래그, DOM `left`/`top` | `#wrap` 설계 좌표 | **`/ factor` 필요** |
| `getPopscaleFactor()` | `script.js` 헬퍼 | 위 `zoom` 값 래핑 |

`common/` 폴더는 수정하지 않습니다. 새 UI 드래그·위치 계산은 `getPopscaleFactor()` 또는 동일한 `zoom` 값을 사용하세요.

### (참고) 1단계 도구 사용법

| 도구 | 조작 |
|------|------|
| 이동 | 빈 곳 드래그 → Pan / 객체·점 드래그 → 이동 (함수 그래프 제외) |
| 점 | 캔버스 클릭 → 점 생성 (A, B, C …) |
| 선분 | 점1 → **마우스 미리보기** → 점2 확정 → 선분 AB |
| 직선 | 점1 → **마우스 미리보기** → 점2 확정 → 무한 직선 dAB |
| 중점 | 점 2개 클릭 → 중점 MAB (보라색, 드래그 불가) |
| 수직이등분선 | 점 2개 클릭 → 수직이등분선 pbAB (청록 점선) |
| 평행선 | 기준 2점 → **미리보기** → 통과점 (`plCAB`) |
| 수직선 | 기준 2점 → **미리보기** → 통과점 (`ppCAB`) |
| 각도 | A → 꼭짓점 B → **마우스 조절** → 확정 (`∠ABC`) |
| 호 | 끝점 A → 끝점 B → **호 위 점 C**로 모양 결정 (`Arc(A,B,C)`) |
| 다각형 | 꼭짓점 순 클릭 → **첫 점 재클릭** 또는 **Enter**로 닫기 (`polyABCD`) |
| 원 | 중심 → **반지름 미리보기** → 확정 (`⊙A`) |
| 삭제 | 점·선분·직선·원 클릭 → 연쇄 삭제 |

2~3단계·4-1·4-2 구현 상세는 Git 이력 및 [`ux_guidelines.md`](ux_guidelines.md) 참고.
